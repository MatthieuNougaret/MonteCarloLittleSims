# MonteCarloLittleSims

Various little Monte-Carlo simulation.

* etimate_pi: estimate $\pi$ value with random sampling in a ] 0, 1 [ square ;
* birthday_problem: explore the birthday problem where the probability to have two (or more) people born the same day increase faster than we can think first ;
* monty_hall: prove the advantage of door switch compare to door keeping in the Monty Hall game.
